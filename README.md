# usd-cad-tracker
Easy to view financial data for viewing on my phone

## Getting Started (Mac)

### 1. Clone the repo
```bash
git clone https://github.com/deanedwardmccall/usd-cad-tracker.git ~/usd-cad-tracker
cd ~/usd-cad-tracker
```

### 2. Run the tracker
```bash
npm start
```
Then open **http://localhost:3000** in your browser.
